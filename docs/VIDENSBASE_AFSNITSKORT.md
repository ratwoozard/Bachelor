## 3) Hvad der er bevidst udeladt i denne opdatering

- Materiale der primært er præsentationsscripts/video-prompts uden tydelig teori-ankring.
- Koblinger der kræver antagelser om data, som ikke fremgår direkte af kilderne.
- Påstande om SoluTalent-performance uden eksplicit reference til artefakt/KPI.

---

## 4) Anvendelsesregel (kort)

Brug dette afsnitskort som opslag:
1. Find først analysebehovet (fx "forklar override").
2. Vælg primær teori og kildehierarki (Tier 1 først).
3. Tilføj mindst én supplerende begrebsforankring fra vidensbasen.
4. Dokumentér med konkret empiri (interview + artefakt eller KPI).
