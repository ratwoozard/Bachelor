# Kvalitet (Lincoln & Guba)

## 1. Kerneide
Kvalitativ troværdighed vurderes via credibility, transferability, dependability og confirmability.

## 2. Tier og kilder
- Tier: 2/method
- Primaer kilde: Lincoln & Guba
- Sekundaer forankring: Kuada (2012); `AKADEMISK_RAMME.md`

## 3. Brug i jeres case
Bruges til at dokumentere kvalitet i interview- og fortolkningsarbejdet.

## 4. Operationalisering og evidenskrav
- Credibility: triangulering + informantnaer dokumentation.
- Transferability: tydelig kontekstbeskrivelse.
- Dependability: transparent metodekæde.
- Confirmability: bias-refleksion og modsigelses-soegning.

## 5. Anti-vaghed
### Det maa I paastaa direkte
- Kvalitet kraever dokumenterede procedurer, ikke kun paastand.
### Det kraever ekstra empiri
- At troværdighed er hoej uden metode- og dataaudit.
### Undgaa-formuleringer
- "Vores data er valide, fordi vi har interviewet eksperter."

## 6. Censorspoergsmaal og svarlinjer
1. Hvordan viser I credibility?  
   Svarlinje: Via triangulering og transparent analyseproces.
2. Hvad med bias?  
   Svarlinje: Bias adresseres eksplicit i metode og diskussion.
3. Kan andre følge jeres proces?  
   Svarlinje: Ja, gennem dokumenterede trin og kriterier.

## 7. Begraensninger
Kriterierne sikrer ikke automatisk stærk analyse uden kritisk anvendelse.
