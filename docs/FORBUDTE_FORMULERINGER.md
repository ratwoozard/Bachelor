# FORBUDTE_FORMULERINGER

Formaal: Eliminere vaghed, overclaiming og metodisk uklart sprog.

## A) Forbudt -> brug i stedet

| Forbudt formulering | Brug i stedet |
|---|---|
| "Dette beviser at..." | "Dette indikerer at..." |
| "AI forbedrer altid..." | "AI kan paavirke ... under disse forudsaetninger..." |
| "Det gælder for alle virksomheder..." | "I denne case observeres ..., hvilket analytisk kan nuancere teori..." |
| "Vi valgte pragmatisme fordi den er bred." | "Vi valgte pragmatisme fordi PF kraever baade maaling og fortolkning." |
| "Vi laver mixed methods fordi vi har tal og citater." | "Vi anvender metodepluralisme med tydelig rollefordeling mellem datakilder." |
| "Informanten viser at..." | "Informanten beskriver at..." |
| "Tallene taler for sig selv." | "Data må fortolkes i kontekst med tydelige definitioner." |
| "Systemet er objektivt." | "Systemets data kan anvendes som evidens under dokumenterede antagelser." |

## B) Forbudte mønstre i metodeafsnit

- Ubegrundet navngivning af metode uden operationel beskrivelse.
- Citering af Tier 3 som om det var Tier 1 metodefundament.
- Omission af begraensninger ved centrale valg.

## C) Forbudte mønstre i analyseafsnit

- Enkeltkilde-konklusioner pa hovedfund.
- Kausal konklusion uden evidenskæde.
- Figur anvendt som evidens uden forklarende tekst.

## D) Hurtig før-publicering check

1. Indeholder teksten ord som "beviser", "altid", "alle", "sikkert"?  
2. Er der tydeligt skel mellem data, fortolkning og implikation?  
3. Er central paastand understoettet af mindst to kilder ved hovedfund?
