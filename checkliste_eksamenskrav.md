# Checkliste – eksamenskrav (Theory of Science)

Brug denne checkliste før aflevering og før mundtlig eksamen.

## 1) Problem og afgrænsning
- [ ] Problemfelt er klart og relevant.
- [ ] Problemformulering er præcis og undersøgbar.
- [ ] Afgrænsning er tydelig (hvad er ikke med?).

## 2) Videnskabsteori
- [ ] Ståsted er tydeligt valgt.
- [ ] Centrale begreber er korrekt defineret.
- [ ] Valg og fravalg af alternative ståsteder er begrundet.

## 3) Metode og empiri
- [ ] Metodevalg matcher videnskabsteoretisk ståsted.
- [ ] Data/empiri er beskrevet klart.
- [ ] Analysefremgang er transparent og logisk.

## 4) Kvalitet i undersøgelsen
- [ ] Kvantitativt spor: validitet/reliabilitet er adresseret.
- [ ] Kvalitativt spor: troværdighed/refleksivitet er adresseret.
- [ ] Begrænsninger og usikkerheder er eksplicitte.

## 5) Etik
- [ ] Informeret samtykke og anonymitet håndteres.
- [ ] Etiske konsekvenser af metodevalg diskuteres.
- [ ] Fremstillingen er fair over for deltagere/data.

## 6) Konklusion og argumentation
- [ ] Konklusion svarer direkte på problemformuleringen.
- [ ] Påstande er understøttet af teori/data.
- [ ] Der sondres klart mellem fund, fortolkning og antagelse.

## 7) Formalia
- [ ] Struktur følger synopsiskrav fra undervisningen.
- [ ] Referencer følger LUSEM-guiden konsekvent.
- [ ] Sprog er præcist, akademisk og uden fyldformuleringer.

## 8) Mundtligt forsvar (egenkontrol)
- [ ] Kan du forklare, hvorfor netop dette ståsted er valgt?
- [ ] Kan du forklare, hvad tilgangen ikke kan forklare?
- [ ] Kan du svare på kritik fra et alternativt ståsted?
