# Agent Quickstart (10 linjer)

1. **Scope:** Kun `staging_imported` → `matched`. Ingen jobsourcing/kontrakt/fakturering. Ingen ML-udvikling.
2. **Kilder:** Tier 1 (Holm, Kuada, Saunders) først ved metode/videnskabsteori; Tier 2 til domæne; Tier 3 kun nuancering.
3. **Pre-go-live:** Ingen rigtige brugere og ingen produktionsdata; skriv det eksplicit ved metode/evidens.
4. **Påstand:** Skal matche `CLAIMS_EVIDENCE_MATRIX` (påstandstype → evidenstype).
5. **Kausalitet:** Brug "kan påvirke" / "indikerer", ikke "beviser" eller "forbedrer" uden forbehold.
6. **Bias:** Nær case / medudvikling → nævn eksplicit og søg modsigende evidens.
7. **Figurer:** Kun efter `FIGUR_REGISTER` / `UNDERVISNINGSFIGURER_KONTEKST`; figur er ikke empirisk bevis.
8. **Sprog:** Undgå formuleringer i `FORBUDTE_FORMULERINGER.md`.
9. **Begreber:** Opslag i `teori_bibliotek/INDEX.md` og `CITATION_MAP.md`.
10. **KPI nu + fuld regelsæt:** Brug kun KPI som måleplan/logging-krav efter go-live (ikke som aktuelle driftresultater), og læs `AGENT_PLAYBOOK.md` før større opgaver.
